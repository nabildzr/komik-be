API untuk mengambil data manhwa dari situs web Komik Station.

Endpoints

  • /api/manhwa-new - Daftar manhwa terbaru
  • /api/manhwa-popular - Daftar manhwa populer
  • /api/manhwa-recommendation - Rekomendasi manhwa
  • /api/genres - Daftar genre
  • /api/genre/:genreId - Manhwa berdasarkan genre
  • /api/search/:searchId - Pencarian manhwa
  • /api/manhwa-detail/:manhwaId - Detail manhwa
  • /api/manhwa-ongoing - Daftar manhwa ongoing
  • /api/chapter/:chapterId - Baca chapter manhwa
